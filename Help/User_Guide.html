1. Quick Start Workflow

  1. Create scenes and characters using the Scenes tab.
  2. Assign actors to characters in the Cast tab.
  3. Set Available Microphones and Max Shares Per Mic.
  4. Use Auto‑Assign or Manual Assign.
  5. Review warnings in Mic Assignments, Explanation, and Timeline.
  6. Adjust per‑scene if needed in the Timeline.
  7. Export final outputs for the booth and backstage.

2. Projects (.mpm2)

Projects store scenes, casting, microphone plans, grouping rules, and timeline overrides.

  • File → New Project
  • File → Open Project
  • File → Save / Save As
Importing scenes or cast via CSV clears any existing microphone plan to prevent mistakes.

3. Scenes Tab

Manual Entry

  • Enter scene name and page range
  • Select characters appearing in that scene
  • Add new characters as needed

CSV Import

  • Top row: scene names
  • Second row: page numbers
  • First column: character names
  • X indicates character appears in scene
If unsure how to format the CSV file, manually enter 1-2 scenes, then select Download Character-Scene CSV button

When CSV is completed, select File > Import Character_Scene_List CSV to import the file.

Scenes without characters are treated as intermissions.
Back to contents

4. Cast Tab

Manual Entry

  • Click Refresh From Project to load all characters that were added on the scenes tab
  • Assign an actor to each character
  • Add new characters as needed by typing their name
  • If actor was already added to previous scene, their name will appear in the dropdown to select for additional characters.

CSV Import

  • First Column: Character Names
  • Second Column: Actor names
Clicking Download Character-Actor CSV button will download a properly formatted CSV file containg all characters added to the Scenes tab and any actors who have already been added to the Cast tab

When CSV is completed, select File > Import Character_Actor_List CSV to import the file.
